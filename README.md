## Installation

From [CocoaPods](http://cocoapods.org):

```ruby
# For latest release in cocoapods
pod "CardStack"

# Want the new new? Use:
pod 'JSNCardStack', :git => 'https://github.com/joenewbry/CardStack.git', :branch => 'develop'
```
